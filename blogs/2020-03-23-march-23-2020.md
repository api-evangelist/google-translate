Change The Cloud Translation Advanced API is migrating to a new quota system that eliminates the distinction between default and maximum limits. This change will roll out gradually, starting on March 30, 2020 and continuing through April 30, 2020 . Since quotas define the usage limits for a project or a user, the change could impact your billing.
