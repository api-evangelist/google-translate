Feature Cloud Translation v3, rebranded as Translation API - Advanced , is now generally available. In addition to the features of Cloud Translation API v3beta1, Translation API - Advanced supports: * User labels * Audit logging * Separate quotas for batch translation and for translation using AutoML models * Client libraries for additional languages PHP, GO, C#, and Ruby Note: Existing users should pay careful attention to client library versions before updating. Translation API - Basic and Translation API - Advanced client libraries are not backward or forward compatible.
